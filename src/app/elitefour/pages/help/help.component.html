Find your favorites without comparing everything one by one

Elite Four narrows a large list by showing a batch of candidates and asking which ones you prefer over the rest.

1

Create a list

Name the topic, choose the maximum batch size, and add every item you want to rank.

2

Add items quickly

Paste multiple lines, import a text file, or add individual items when you need more control.

3

Pick from each batch

Select one or more items you like better than the unselected items, then confirm the round.

How the picker works

Each round shows up to the configured batch size. When you choose items and confirm, every unselected item in that batch is eliminated by the selected items. When only one non-eliminated item remains, it becomes the next favorite.

After a favorite is found, the items it eliminated return to the pool so the next favorite can be chosen correctly. You can stop once you have enough favorites or keep going until the entire list is ordered.

The original algorithm idea comes from Favorite Pokemon Picker.

Useful details

  • The list overview is where you create, search, and open lists.
  • The item screen is where you add items, paste several at once, import text files, and export results.
  • You cannot edit item names once picking has started unless you reset the algorithm.
  • Skip batch is equivalent to selecting every item in the current batch.
  • Spotify is optional and only adds matching/export helpers for music lists.
